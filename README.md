  - **Stacks utilisés**: Symfony, Docker, Mysql, React, ChakraUi, Api The Movie database

  - **Résumé**: Le AirBnB du luxe.
  - **Stacks utilisés**: React, TypeScript, TypeORM, GraphQL, Next.js, PostgreSql Docker, ChakraUi
